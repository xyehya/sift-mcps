# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it using [GitHub's private vulnerability reporting](https://github.com/AppliedIR/opensearch-mcp/security/advisories/new) rather than opening a public issue.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Supported Versions

| Version | Supported |
| ------- | --------- |
| main    | Yes       |

## Security Considerations

This tool is designed for use in forensic analysis environments. Users should:
- Run in isolated/sandboxed environments when analyzing untrusted data
- Keep dependencies updated
- Use individual OS accounts rather than shared accounts for audit accountability
